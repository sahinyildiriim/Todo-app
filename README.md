# Todo App

A simple task management application (To-Do App) to easily track your daily tasks. Built with a React frontend and Node.js/Express backend.  

---

## Features

- Add, edit, and delete tasks  
- Mark tasks as completed or incomplete  
- User authentication (Login / Register)  
- Responsive design for mobile and desktop  
- Simple and user-friendly interface  

---

## Technologies

- **Frontend:** React, React Router, Styled Components  
- **Backend:** Node.js, Express, MongoDB (or preferred database)  
- **Others:** Axios, JWT (for authentication), bcrypt (for password encryption)  

---

## Installation

1. Clone the repository:

git clone https://github.com/sahinyildiriim/Todo-app.git
```bash
cd todo-app

2.Install backend dependencies: 
cd backend
npm install

3-Install frontend dependencies:
cd ../frontend
npm install

## 4.Create a .env file and add necessary variables (e.g., MongoDB URI, JWT_SECRET).

5.Run backend and frontend in separate terminals:

# Backend
cd backend
npm start

# Frontend
cd frontend
npm start

Usage:

Open the app and register or login.
Add new tasks using the form.
Mark tasks as completed or delete them when done.
Use the edit button to modify existing tasks.
